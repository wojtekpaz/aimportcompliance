What this does

The GRI rules of interpretation allow a product to be described in more than one legally defensible way, depending on which attribute is foregrounded — material, function, state of presentation, essential character, or principal use.

This tool identifies those alternative framings and runs each one through the deterministic GRI classification engine. Only alternatives that the engine classifies to a declarable code appear below. The AI proposes framings; the engine decides legal validity.

The GRI path and legal basis for each alternative are the substantive output. Duty differences are shown as a consequence, not the objective.
